# 10. 发布与分发

### 10.1 分发渠道

- **GitHub Releases**：主要分发渠道
  - 提供 `.exe` 安装包（Tauri NSIS installer）
  - 提供 `.msi` 安装包（可选）
  - 提供便携版 `.zip`（免安装）

### 10.2 发布产物

| 文件 | 说明 |
|------|------|
| `qq-auto-like-plus_x.y.z_x64-setup.exe` | Windows x64 安装包 |
| `qq-auto-like-plus_x.y.z_x64_portable.zip` | 便携版 |
| `CHANGELOG.md` | 更新日志 |

### 10.3 版本策略

- 遵循语义化版本 (SemVer)：`MAJOR.MINOR.PATCH`
- V1.0.0：核心功能完整可用
- V1.x.0：增量功能迭代
